# Flexa - Second-Hand Shoes Marketplace

![Flexa Logo](public/images/logo.png)  

**Flexa** is a modern online marketplace for **second-hand shoes**, offering sustainable, high-quality, and affordable footwear. Our platform allows users to browse, shop, and order carefully curated pre-loved shoes while promoting eco-friendly fashion.  

This project is currently in **alpha stage**, meaning it's functional but still under active development and improvements are ongoing.
